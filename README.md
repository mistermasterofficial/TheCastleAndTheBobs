# The Castle And The Bobs
![](./assets/banner2.png)
_It's a small pseudo-3d game about Bob._

For now its supporting only russian language, but you can edit localization in settings ("settings/loc.json")
Also you can change a music and sounds volumes, sensitivity and others ("settings/general.json")
